# 经济法刷题

基于刘大洪《经济法》第三版的期末复习刷题工具，适配手机浏览器。

## 功能

- 📝 **刷题模式** — 按章节/题型筛选，单选+多选+判断共 667 题
- 📘 **错题本** — 自动收集错题，支持错题重刷
- ⭐ **收藏夹** — 收藏重点题目，分类回顾
- 📋 **背诵资料** — 名词解释、简答题、案例分析共 133 条
- 📤 **学习记录备份** — 导出/导入做题进度、错题本、统计

## 使用方式

### 手机访问（推荐）

部署到 GitHub Pages 后，手机浏览器打开 `https://Flash749.github.io/jingjifa/` 即可使用。

### 本地打开

下载 `index.html`，用浏览器直接打开。

## 题库信息

- 教材：刘大洪《经济法》第三版
- 总题量：667 道（单选 316 + 多选 196 + 判断 152 + 背诵 133）
- 覆盖章节：第一章 ～ 第十八章

## 技术

纯静态 HTML，单文件包含全部题库数据和交互逻辑，无需后端。
